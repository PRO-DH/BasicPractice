데이터베이스 = 데이터 + 베이스 
DBMS = Database Management System
데이터 - 원석 / 특정 목적을 평가하거나 정제하지 않은 값, 사실 또는 자료 그 자체
정보 - 보석 / 수집한 데이터를 어떠한 목적을 위해 분석하거나 가공하여 가치를 추가하거나 새로운 의미를 이끌어 낼 수 있는 결과

가치 있는 정보를 얻기 위해서는 데이터를 효율적으로 수집, 통합하고 체계적으로 관리, 분석해야한다.
효율적인 관리와 검색을 위해 구조화한 데이터 집합을 데이터베이스 라고 한다.

응용프로그램이 필요한 데이터 작업을 DMBS에 요청하면 DBMS는 자신이 관리하는 데이터베이스로 관련 작업을 수행하고 결과값을 제공한다.

SQL - RDBMS에서 데이터를 다루고 관리하는데 사용하는 데이터베이스 질의 언어이다.
